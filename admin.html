📖 使用說明

  1. 準備好 questions.xlsx 檔案（可從後台「匯出 XLSX」取得範本）
  2. 點擊上方上傳區域選擇檔案
  3. 確認題目數量無誤後，點擊「上傳到 GitHub」
  4. 系統會自動透過 Cloudflare Worker 上傳，並觸發 GitHub Actions
  5. 約 1-2 分鐘後題庫更新生效，學生端自動載入最新題庫

⚠️ 注意：上傳後請等待 GitHub Actions 完成處理，期間學生可能看到舊題庫。
