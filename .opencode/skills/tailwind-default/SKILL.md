## What I do
- When creating any HTML/CSS page or presentation, always use Tailwind CSS by default
- For simple pages, use the CDN version: `<script src="https://cdn.tailwindcss.com">`
- For complex projects, install via npm: `npm install tailwindcss`
- For presentations, combine Tailwind CSS with Reveal.js or Slidev

## When to use me
Always active. Apply Tailwind CSS by default to all HTML/CSS work.

## Guidelines
- Use Tailwind utility classes directly in HTML (no custom CSS unless necessary)
- Keep custom CSS minimal; prefer Tailwind utilities
- Use responsive prefixes (sm:, md:, lg:, xl:) for responsive design
- Use dark: variant for dark mode support when applicable
