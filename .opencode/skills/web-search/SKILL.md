## What I do
- Search the web for up-to-date information using DuckDuckGo (no API key required)
- Fetch and read the content of web pages
- Find documentation, news, tutorials, and answers to technical questions

## When to use me
Use this skill whenever the user asks a question that requires:
- Current or real-time information (e.g. latest news, prices, events)
- Information that may be beyond the AI's training data cutoff
- Looking up documentation, APIs, libraries, or code examples
- Verifying facts, finding sources, or researching topics
- Reading the content of a specific URL

## How to invoke
Simply mention you need to search for something, and I will use the web-search and fetch MCP tools automatically.
