## What I do
- Read files in the project directory using the filesystem MCP server
- Recursively list directory contents
- Read any text-based file (code, config, documentation, etc.)
- Navigate the project structure to find relevant files

## When to use me
Use this skill whenever:
- The user asks about project structure or code organization
- You need to read code files to understand implementation
- You need to check configuration files, package definitions, etc.
- The user asks you to analyze, modify, or debug code in the project

## How to invoke
When you need to read files, use the filesystem MCP tools to list directories and read file contents. Always check the project structure first before diving into specific files.
